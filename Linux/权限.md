### 权限

```shell
-rw-rw-r-- 1 python python  43  5月  9  05:56 01.py
drwxrwxr-w 2 python python 313  5月  4  14:49 demo
释义：
第一列：-代表文件  ，d 代表目录
第一组 rwx :代表当前用户的权限
第二组 rwx :代表当前用户组的权限
第三组 r-w :代表其他用户的权限
1/2 ：代表硬链接数。有多少种方式，可以访问到当前目录或文件
第一列 python :代表当前用户为 python
第二列 python :代表当前的用户组为 python
```

### chmod 修改文件权限

chmod +/- rwx 文件名/目录

chmod -R 755 文件名/目录名

### chown 修改文件拥有者

chown  用户名  文件名/目录名

### chgrp 修改文件目录的组

chgrp -R 组名   文件名/目录名
