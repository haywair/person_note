### python 基础

---

1. 单行注释： # 
2. 在代码后添加注释，至少有两个空格后再加 # 
3. 多行注释 ： 一对连续的三个引号（可以是单引号，也是双引号）
4. python pep8 规范文档
5. 算术运算符

```
+ 加
- 减
* 乘
/ 除
// 取整除
% 取余数
** 幂运算
```

6. 字符串拼接

```
a = 'ffh' * 3
print(a)  # 输出 ffhffhffh 
```

7. 数据可以分为数字型和非数字型：

```
数字型：
整型
浮点型
布尔型: true 非0  false 0
复数型（主要用于科学计算 例如：平面场问题、波动问题、电感电容问题）

非数字型：
字符串
列表
元祖
字典
```

8. 使用type查看变量类型（主要用于ipython等交互方式）
